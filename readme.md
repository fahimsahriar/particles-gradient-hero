# Particle Gradient Animation

A lightweight animated background built using **Particle.js** combined with a smooth **CSS gradient** effect.
This setup is ideal for landing pages, hero sections, or any UI element where a modern interactive background is required.

## Features

* Particle.js-powered particle system
* Layered animated gradient background
* Responsive and smooth performance
* Easy to integrate with any HTML, WordPress, or frontend project

## Technologies Used

* **Particle.js** for particle system
* **CSS gradients** for background animation
* **JavaScript** for initialization and configuration

## Why I built this

To create a visually appealing background effect that feels modern and interactive. This component can be reused across projects to enhance UI/UX without relying on heavy libraries.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/yourusername/particles-gradient-hero.git
```

2. Open `index.html` in your browser or integrate the effect into your project.

3. Customize particle settings inside:

```
app.js
```
